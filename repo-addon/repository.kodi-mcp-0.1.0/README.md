# Kodi MCP Server Repository

This repository contains packages for the Kodi MCP Server project.

## Contents

- Backend server tools
- CLI wrappers (when implemented)
- Bridge addons for remote integration

## Installation

1. Install this repository addon via "Install from zip" in Kodi
2. Navigate to the repository from the Add-ons menu
3. Install desired packages from available addons
